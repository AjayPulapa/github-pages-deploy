# Getting Started

> Set up your GitHub Pages site in under 5 minutes.

## Prerequisites

- A GitHub account
- Git installed locally
- Basic knowledge of HTML/CSS

## Installation

Clone or fork this repository to get started:

```bash
git clone https://github.com/yourusername/yourrepo.git
cd yourrepo
```

## Configuration

Edit the `index.html` and pages inside `/pages/` to match your content. Update the nav links and sidebar links to point to your own pages.

> 💡 All pages share `assets/css/style.css` and `assets/js/main.js` — edit once, applies everywhere.

## Deploy to GitHub Pages

1. Push your code to GitHub: `git push origin main`
2. Go to your repo → **Settings → Pages**
3. Set source to **main branch / root folder**
4. Your site is live at `https://username.github.io/repo/`
